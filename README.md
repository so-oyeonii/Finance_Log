# Finance Log

> 내 돈의 흐름을 한눈에. AI가 함께하는 스마트 가계부.

---

## 왜 Finance Log인가요?

돈 관리, 하고 싶은데 엑셀은 귀찮고 앱은 복잡하고...

**Finance Log**는 가계부 + 자산관리 + 투자 포트폴리오 + AI 재무 멘토를 하나에 담았습니다.
인터넷 없이도 동작하고, 내 데이터는 내 브라우저에만 저장됩니다.

---

## 이런 분께 딱이에요

- **대학원생** — BK인건비, 조교수당, 장학금... 불규칙한 수입을 깔끔하게 정리하고 싶은 분
- **직장인** — 월급 관리부터 IRP, 주식 포트폴리오까지 한 곳에서 보고 싶은 분
- **재테크 입문자** — 예적금 만기수령액, 주식 수익률을 자동으로 계산받고 싶은 분

---

## 주요 기능

### 자산 한눈에 보기
은행별 계좌 잔액, 예적금 만기수령액, 투자 수익률을 한 화면에서 확인하세요.

### 똑똑한 거래 입력
`"카카오뱅크로 스타벅스 5000원"` — 이렇게 말하면 AI가 알아서 분류합니다.
영수증 사진을 찍으면 자동으로 입력되기도 해요.

### 투자 포트폴리오
국내주식, 미국주식, ETF, 코인까지. 평균단가, 수익률, 배당금을 자동 계산합니다.

### AI 재무 멘토
"이번 달 어디서 돈을 많이 썼지?" — 내 소비 패턴을 분석하고 맞춤 조언을 해줍니다.

### 모드 전환
대학원생 모드와 회사원 모드를 원클릭으로 전환. 카테고리, 색상, AI 성격이 모두 바뀝니다.

### 오프라인 우선
모든 데이터는 브라우저 내장 DB에 저장됩니다. 인터넷 없이도, 로그인 없이도 사용 가능합니다.

## 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Database | IndexedDB (Dexie.js) — Local-first |
| State | Zustand, React Hook Form + Zod |
| AI | OpenAI GPT-4o |
| Charts | Recharts |
| 기타 | PWA 지원, Lucide Icons, dnd-kit |

## 시작하기

```bash
# 의존성 설치
npm install

# 환경변수 설정
cp .env.example .env.local
# .env.local에 OPENAI_API_KEY 입력

# 개발 서버
npm run dev
```

http://localhost:3000 에서 확인하세요.
